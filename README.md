# ⚖️ LexControl AI

Sistema inteligente de asistencia jurídica que analiza eventos procesales y genera recomendaciones automáticas utilizando inteligencia artificial.

---

## 📌 Descripción

**LexControl AI** es una aplicación diseñada para optimizar la gestión de causas legales, ayudando a abogados y profesionales del derecho a:

- Interpretar eventos procesales
- Calcular plazos estimados
- Sugerir acciones legales
- Determinar niveles de urgencia

Todo esto a partir del ingreso de texto en lenguaje natural.

---

## 🚀 Funcionalidades principales

- 🧠 **Análisis con IA**  
  Interpretación automática de eventos procesales mediante modelos de lenguaje.

- ⏱️ **Cálculo de plazos**  
  Estimación de vencimientos según el contexto ingresado.

- 📋 **Recomendaciones legales**  
  Sugerencias de acciones a seguir basadas en el análisis.

- 🚨 **Nivel de urgencia**  
  Clasificación del evento según prioridad.

- 💬 **Entrada en lenguaje natural**  
  No requiere formato técnico estructurado.

---

## 🧠 Integración de IA

Se utiliza un modelo de lenguaje (Gemini API) para:

- Procesar texto jurídico
- Generar respuestas estructuradas
- Simular razonamiento legal básico

## ⚙️ Instalación

1. Clonar el repositorio:

```bash
git clone https://github.com/carocook/LexControl_IA.git
cd LexControl_IA
```

2. Crear entorno virtual:

```bash
python -m venv venv
source venv/bin/activate  # Mac/Linux
venv\Scripts\activate     # Windows
```

3. Instalar dependencias:

```bash
pip install -r requirements.txt
```

4. Confirgurar variables de entorno

```bash
API_KEY=tu_api_key_de_gemini
```

5. Ejecutar la aplicacion:

```bash
python main.py
```
